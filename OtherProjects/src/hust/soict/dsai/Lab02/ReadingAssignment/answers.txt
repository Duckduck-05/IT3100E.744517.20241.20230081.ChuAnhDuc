Quesion: When should accessor methods be used?

Answer:
1. When we need to hide the internal representation of an object and provide a more abstract interface to it
2. When we need to provide read-only access to a property of attribute
3. When we need to enforce some sort of invariant or validation rule on the value being set

Question: If you create a constructor method to build a DVD by title then create a constructor method to build  a DVD by category. Does JAVA allow you to do this?  

Answer: Yes, Java allows to create multiple constructor methods for a class, as long as they have different parameter lists. This is known as constructor overloading.
